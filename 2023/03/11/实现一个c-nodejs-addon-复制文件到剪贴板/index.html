背景

在 Electron 聊天应用中，有时需要进行文件操作，而 Electron 提供的一些 API 无法完全满足需求，因此使用 C++ 原生实现的轮子，并将其封装成 Node.js addon ，再使用 Electron Builder 进行本地模块的编译，以供应用内部调用，这是一个很好的解决方案，以下 c++ 代码是网上找的轮子，自己通过 node-addon-api 封装一下，mark过程
